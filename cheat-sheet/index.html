HTML-Specific

Tag Name Description
<head> Head The wrapper element for the metadata of a html file.
<body> body Stores the ACTUAL information and data for the viewer.
<H1-H6> Header Headers are title-sized text information. They range from h1 to h6, shrinking in size.
<p> Paragraph The paragraph is a block of written text that outputs to the user. You can write as much as you want in here, but any space between will be ignored if not used with <br>
<pre> Pre-Paragraph Almost identical to paragraph however it holds the spaces. it will print exactly how it is written.
<q> Quotations Hold's short quotations.
<Div> Division Division in HTML that can be given a class/id so it can be edited separately in CSS
<img> Image stores the link to an image that appears in your HTML
<input> Input Creates an input section in your code for user's to add or enter information.
<img> Image stores the link to an image that appears in your HTML
<section> Section Creates a linkable section in the HTMl that you can then use navbar for
<ul/or> Unordered/Ordered List An (un)ordered list of items.
<li> Listed items The items stored in ul/ol

CSS SPECIFIC

Tag Name Description
font-family: Font Family Changes the font of the section of CSS
color: Color Changes the color of the text
background-color: Background Color Changes the color of the background of the object (acts as a high lighter for text, acts as the bg for a container)
text-align: Text Align Allows you to change alignment from, Center, left, right
border: Border Allows you to add a border, make it's shape (dotted, solid...), and width'
padding: Padding Adds space between the contents
